تطوير تطبيقات شوبيفاي

Wishlist Pro

تطبيق شوبيفاي متكامل يوفر للتجار حلاً أصلياً وعالي الأداء لقائمة الأمنيات مبني بأحدث تقنيات الويب.

تطبيق شوبيفاي React / Polaris Node.js / GraphQL
الدور مطور تطبيقات Full-stack
الجدول الزمني 8 أسابيع
مجموعة التقنيات React · Node · GraphQL · Prisma
النتيجة الرئيسية تطبيق SaaS جاهز للإنتاج

التحدي

يعاني العديد من التجار من تطبيقات قوائم الأمنيات التابعة لجهات خارجية والتي تكون ثقيلة، وصعبة التخصيص، وتؤثر سلباً على أداء المتجر بسبب أكواد الطرف الثالث البرمجية. كان الهدف هو بناء تطبيق شوبيفاي يبدو كجزء أصلي من المنظمة، يندمج بعمق في نظام شوبيفاي مع الحفاظ على أداء عالٍ.

النهج المتبع

قمت ببناء التطبيق باستخدام قالب تطبيق شوبيفاي الرسمي، مع الاستفادة من React Router لتجربة لوحة التحكم المدمجة و Node.js للخلفية. تم بناء واجهة المستخدم بالكامل باستخدام Polaris، مما يضمن أنها تبدو كجزء أصلي من لوحة تحكم شوبيفاي.

  • لوحة تحكم مدمجة مبنية باستخدام مكونات Shopify Polaris و App Bridge.
  • خلفية مدعومة بـ Node.js و Prisma ORM لإدارة فعالة للبيانات.
  • استخدام Shopify GraphQL Admin API لمزامنة بيانات المنتجات والعملاء.
  • تكامل مع واجهة المتجر عبر App Embed Blocks لتجربة تاجر لا تتطلب أي كود.

النتيجة

Wishlist Pro هو تطبيق شوبيفاي جاهز للإنتاج يوفر تجربة سلسة لكل من التجار والعملاء. يوفر تكاملاً عميقاً مع ميزات شوبيفاي الأصلية ويزود التجار برؤى قيمة حول نية العملاء من خلال تحليلات بيانات قوائم الأمنيات.
