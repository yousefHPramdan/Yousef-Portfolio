## Repo 3: arabic-rtl-shopify

```markdown
# Arabic RTL Shopify — Complete Guide

The definitive guide to building production-quality Arabic/English bilingual Shopify stores.

## What Most Guides Get Wrong
Most "Arabic Shopify" tutorials just add `direction: rtl` to the body.
This breaks: icon directions, number alignment, animation directions,
third-party app layouts, checkout, and font rendering.

## The Right Way

### 1. HTML lang attribute
<html lang="ar" dir="rtl">  or  <html lang="en" dir="ltr">
Use Shopify's request.locale to switch dynamically.

### 2. Arabic Fonts
Recommended: Almarai (all weights), Tajawal, Cairo
Self-host using @font-face. Never load from Google Fonts in production.

### 3. Logical CSS Properties
Use logical properties instead of left/right:
- margin-inline-start instead of margin-left  
- padding-inline-end instead of padding-right
- border-inline-start instead of border-left
These flip automatically in RTL.

### 4. Icon Direction
SVG chevrons, arrows need to be flipped in RTL:
.icon-arrow { transform: scaleX(var(--icon-dir, 1)); }
html[dir="rtl"] { --icon-dir: -1; }

### 5. GSAP Animations
Always start: x percentages instead of pixel values.
gsap.from('.hero', { x: '100%' }) → slides from right in LTR, left in RTL = correct.
gsap.from('.hero', { x: 100 }) → always from the right = WRONG in RTL.
```
