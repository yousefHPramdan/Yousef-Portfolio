# Yousef Ramadan — Front-End Shopify Developer

> **Portfolio Website** | [youseframadan.me](https://youseframadan.me)

A premium, custom-coded portfolio for a Shopify Developer specializing in high-performance storefronts, Shopify Plus migrations, and custom app development. The site features a unique dark-mode design with glassmorphism, interactive elements, and detailed case studies of e-commerce projects.

## 🚀 Key Features

- **Premium Design**: Custom dark-mode theme with glassmorphism, micro-interactions, and smooth animations.
- **Interactive Elements**:
  - **3D Tilt Effects**: Cards and images react to mouse movement.
  - **Magnetic Buttons**: CTA buttons follow the cursor.  
  - **Cursor Effects**: Custom cursor with a magnetic ring.
  - **Scroll Animations**: Elements fade in and slide up on scroll.
- **Responsive & Accessible**:
  - Fully responsive layout (mobile-first).
  - Skip-to-content link for accessibility.
  - ARIA labels and proper semantic HTML.
- **Performance Optimized**:
  - Lazy loading for images.
  - Optimized asset delivery.
  - Fast page transitions.

## 🛠️ Tech Stack

- **Core**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**:
  - **Tailwind CSS**: Utility-first CSS framework for rapid development.
  - **Custom CSS**: Custom properties and animations for unique effects.
- **Fonts**:
  - **Syne**: For headings and display text.
  - **Plus Jakarta Sans**: For body text.
  - **JetBrains Mono**: For code snippets and technical details.
- **Icons**: Font Awesome
- **Deployment**: Static hosting (GitHub Pages compatible)

## ⚡ Getting Started

1.  **Clone the repository**:
    ```bash
    git clone <repository-url>
    cd Yousef-Portfolio
    ```

2.  **Open in browser**:
    Simply open `index.html` in your web browser.

3.  **Development**:
    If you want to modify the site, you can use any static site server (like VS Code's Live Server extension) or deploy it to a hosting provider.
