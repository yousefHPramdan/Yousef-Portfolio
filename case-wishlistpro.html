The Challenge

Most wishlist apps on the Shopify App Store suffer from heavy script bloat that slows down merchant storefronts. Wishlist Pro was conceived as a "performance-first" alternative that provides a premium, native feel without the typical frontend performance penalty.

The Approach

I architected the app using a React (Remix) backend for the merchant admin, utilizing Shopify's Polaris design system for a native experience. The frontend injection was built using vanilla JS and custom Liquid snippets to ensure zero external dependencies on the storefront.

I implemented a scalable NoSQL database architecture to handle high-frequency wishlist interactions across multiple storefronts simultaneously, ensuring real-time syncing and high availability.

The Result

A production-ready Shopify App that delivers a fluid, lightning-fast wishlist experience. The app demonstrates my ability to handle complex full-stack Shopify integrations, from deep-level API authentication to high-performance frontend engineering.
